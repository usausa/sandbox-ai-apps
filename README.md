# AI利用アプリケーションサンプル

## [チラシ校閲](./llm-flyer)

### 概要

チラシ画像をアップロードし、登録されているマスタデータの値段と突合・比較を行なうアプリケーション。

### 使用技術

- LLM
- ベクトル検索

## [調査員不正入力検出](./llm-inspector)

### 概要

調査員が入力した計測データをアップロードし、入力値のパターンから実測値以外の入力を検出するアプリケーション。

### 使用技術

- LLM

## [調査員足順最適化](./llm-route)

### 概要

電気設備点検の調査員が、その日の訪問先一覧から1日・調査員1名分の足順（訪問順）を決定するアプリケーション。移動時間と指定時間帯を考慮した足順をアルゴリズムで算出し、その案をLLMでルール検証・改善提案するハイブリッド構成。

### 使用技術

- LLM

## [POSレジ操作不正検出](./llm-pos)

### 概要

小売店の従業員によるPOSレジ操作の不正（取消抜き取り・返品抜き取り・ポイント不正など）を、トランザクションログから検出するアプリケーション。トランザクションを決定論的に集計し、その結果と生データをLLMに渡して「1営業日×1従業員」単位で不正確率を判定するハイブリッド構成。

### 使用技術

- LLM

## 共通仕様

- C# 10.0 / Blazor Serverアプリケーション
- Microsoft Foundry使用
